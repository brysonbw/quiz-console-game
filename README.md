# Quiz Console Game
Simple java quiz console game
- 5 questions relating to java
- Displays score and percentage after all questions answered

## Usage
```shell
git clone git@github.com:brysonbw/quiz-console-game.git
```

### Go to project dir
```shell
cd quiz-console-game/src
```

### Compile
```shell
javac *.java
```

### Run
```shell
java Main
```
